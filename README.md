# Legal-GenAI-RAG-Enhanced-Legal-Judgement-Summarization-using-LLaMA-2-QLoRA
An end-to-end Generative AI pipeline designed to automate the summarization of complex Indian legal judgements. This project leverages the IN-Ext dataset to provide both full-length and segment-wise summaries, addressing the challenge of processing long-form legal documents through Large Language Models (LLMs).
